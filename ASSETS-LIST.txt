========================================
QUICK TIPS
========================================

1. NAMING: Names are case-sensitive. Use exact names as shown.

2. FORMATS:
   - Images: PNG, JPG, or JPEG
   - Videos: MP4 (H.264 codec recommended)

3. SIZES: Dimensions shown are recommended. Maintain aspect ratios.

4. OPTIMIZATION:
   - Compress images to reduce file size
   - Keep videos under 10MB when possible
   - Use progressive JPEGs for faster loading

5. FALLBACKS:
   - Site works without images/videos (shows placeholders)
   - Add assets as you create them
   - Placeholders clearly labeled

6. BATCH RENAMING:
   - If you have existing assets, batch rename them
   - Many tools available for bulk file renaming
